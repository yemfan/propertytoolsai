# Scripts

| Script | Purpose |
|--------|---------|
| `vercel-monorepo-root-build.mjs` | Vercel repo-root build: clean `.next`, `turbo build`, optional `NEXT_DIST_IN_MONOREPO_ROOT` |
| `smoke-db-all.mjs` | Run Supabase migration smoke tests across `leadsmartai` and `propertytoolsai` |
| Other `*.mjs` | App-specific tooling (SEO, sync, etc.) |

Install and run from **repo root** with **pnpm** (`pnpm run …`).
