# Static images

- **`lslogoh.png`** — main header/footer logo (`LeadSmartLogo`). Sync from repo root `Images/LSLogoH.png` when updating art.
- **`ptlogo.png`** — legacy / unused if LeadSmart uses `lslogoh.png` only.
- **`lslogo.png`** / **`ls32.png`** / **`ls180.png`** — optional; favicons in `app/layout.tsx` expect the `ls*` files when you add them.
